# Muel Portfolio Website

프로그래머 Muel의 개인 포트폴리오 웹사이트입니다. 

🌐 **Live Demo:** [muel.kro.kr](https://muel.kro.kr)

## 📖 소개
파이썬과 웹 디자인 및 개발(HTML, CSS, JS 등)을 중점적으로 다루며, 인공지능 분야를 탐구하는 개발자 Muel의 포트폴리오입니다. 현재 404.team의 팀장으로 활동하고 있습니다.

## ✨ 주요 기능
* **다크 모드 (Dark Mode):** 우측 사이드바의 🌙 버튼을 통해 부드러운 다크 모드 전환을 지원합니다.
* **손전등 효과 (Flashlight Effect):** 💡 버튼 클릭 시 마우스 커서를 따라다니는 인터랙티브한 조명 효과를 제공합니다.
* **글래스모피즘 사이드바 (Glassmorphism Sidebar):** 우측 하단에 위치한 반투명한 플로팅 메뉴를 통해 GitHub 및 이메일로 빠르게 이동할 수 있습니다.
* **인터랙티브 툴팁:** 경력 및 프로젝트 목록에 마우스를 올리면 상세 설명, 외부 링크 및 사용 기술 스택 태그가 나타납니다.

## 🛠 기술 스택
* **Frontend:** HTML5, CSS3, Vanilla JavaScript
* **Font:** GriunPolFairness
